                                                      🛒 E-Commerce Website 
A full-stack E-Commerce Web Application built using Django. This project allows users to browse products, add items to cart, place orders, and interact with a modern user-friendly interface.


📌 Features
  -> 👤 User Features
        User Registration & Login
        Secure Authentication System
        Browse Products
        Search Products
        Add to Cart
        Update Cart Quantity
        Place Orders
        View Order History


  ->🛍️ Product Features
        Product Listings with Images
        Product Details Page
        Categories / Filtering
        Price Display

  ->🛠️ Admin Features
        Add / Update / Delete Products
        Manage Users
        View Orders
        Admin Dashboard (Django Admin Panel)

🧑‍💻 Tech Stack
   Backend: Django (Python)
   Frontend: HTML, CSS, JavaScript
   Database: SQLite (default)
   Media Handling: Django Media Files
   Authentication: Django Auth System

Bash
Copy code

cd ecommerce-project
2️⃣ Create Virtual Environment
Bash
Copy code
python -m venv venv
venv\Scripts\activate   # Windows
3️⃣ Install Dependencies
Bash
Copy code
pip install -r requirements.txt
4️⃣ Apply Migrations
Bash
Copy code
python manage.py makemigrations
python manage.py migrate
5️⃣ Create Superuser
Bash
Copy code
python manage.py createsuperuser
6️⃣ Run Server
Bash
Copy code
python manage.py runserver
7️⃣ Open in Browser
Copy code
http://127.0.0.1:8000/
